# PhishGuard



**PhishGuard ** is a cutting-edge **AI-powered phishing detection and cybersecurity platform** designed to protect users from malicious websites, emails, and social engineering attacks. Built with **React, Vite, and Tailwind CSS**, it offers an interactive, modern, and user-friendly interface.  

---

##  Live Demo
[Click here to visit the live site](https://phishiguardai.netlify.app/)

---

##  Features

- **AI-Powered Phishing Detection:** Detect malicious links, phishing emails, and fake websites instantly.  
- **Real-Time Analysis:** Fast and accurate analysis using modern frontend technologies.  
- **Educational Insights:** Learn about common phishing tactics and cyber threats.  
- **Responsive Design:** Works flawlessly on mobile, tablet, and desktop.  
- **Beautiful UI:** Modern and clean interface with animations and informative visuals.  


##  Tech Stack

- **Frontend:** React + Vite  
- **Styling:** Tailwind CSS + Custom Design System  
- **Icons & UI Components:** lucide-react + shadcn/ui  
- **State Management:** React Hooks (useState, useEffect)  
- **Hosting:** Vercel

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/devbyaadi/PhishGuard.git
   cd PhishGuard
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open your browser**
   ```
   Navigate to http://localhost:5173 (or the port shown in your terminal)
   ```
